# Windows 截图工具 - 快速开始

## 🚀 最简单的运行方法

### 方法 1：双击运行（最简单）

**只需要 1 个文件：**
```
启动截图工具.bat
screenshot_simple.py
```

**操作：**
1. 双击 `启动截图工具.bat`
2. 程序会自动检查并安装依赖
3. 自动启动截图工具

---

### 方法 2：直接运行 Python 脚本

**前提**：已安装 Python 3.8+

**步骤：**

```bash
# 1. 安装依赖（只需一次）
pip install PyQt5 mss Pillow

# 2. 运行程序
python screenshot_simple.py
```

---

### 方法 3：打包成 EXE（推荐分发）

**优点**：无需 Python，双击运行

```bash
# 1. 安装打包工具
pip install pyinstaller

# 2. 打包
pyinstaller simple.spec

# 3. 运行 EXE
dist\ScreenshotTool.exe
```

---

## 📸 使用说明

### 运行程序后：

1. **选择区域**
   - 屏幕变暗，出现黑色半透明窗口
   - 按住鼠标左键拖动，绘制红色矩形
   - 松开鼠标完成选择

2. **预览和保存**
   - 自动显示悬浮预览窗口
   - 拖动窗口到合适位置
   - 点击"💾 保存"保存截图
   - 点击"✖ 关闭"退出程序

## ❓ 常见问题

### Q: 提示"未找到 Python"

**A**: 需要安装 Python
- 下载：https://www.python.org/downloads/
- ⚠️ 安装时勾选 "Add Python to PATH"

### Q: 程序一闪而过

**A**: 正常现象，程序启动后会隐藏控制台窗口
- 如果出错，会显示错误信息
- 使用 `启动截图工具.bat` 可以看到详细错误

### Q: 想要独立的 EXE 文件

**A**: 运行 `build_simple.bat` 打包
- 打包后的 EXE 在 `dist\ScreenshotTool.exe`
- 可以复制到其他电脑直接运行
