# 用户安装指南

## Windows 用户

### 最简单的使用方式 ✅

**直接双击运行 `ScreenshotTool.exe` 即可！**

---

### 如果程序无法运行：

#### 1. 安装 Visual C++ 运行时

下载并安装：[Microsoft Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)

- 文件大小：约 14MB
- 安装时间：1 分钟
- 重启电脑：无需重启

#### 2. 检查 Windows 版本

| Windows 版本 | 兼容性 | 说明 |
|--------------|--------|------|
| Windows 11 | ✅ 完美兼容 | 无需额外安装 |
| Windows 10 | ✅ 完美兼容 | 无需额外安装 |
| Windows 8.1 | ⚠️ 需要安装 VC++ 运行时 | 下载上面链接 |
| Windows 7 | ⚠️ 需要安装 VC++ 运行时 | 下载上面链接 |
| Windows XP | ❌ 不支持 | 请升级系统 |

#### 3. 被杀毒软件拦截？

这是误报！可以：

1. **临时关闭杀毒软件**
2. **添加到白名单**：
   - Windows Defender：设置 → 病毒和威胁防护 → 管理设置 → 排除项
   - 第三方杀毒软件：类似设置

## macOS 用户

macOS 版本需要从源码运行：

```bash
# 1. 安装 Python 3（如果未安装）
# 从 https://www.python.org/downloads/ 下载安装

# 2. 安装依赖
pip3 install -r requirements.txt

# 3. 运行程序
python3 win1.py
```

---

## 常见问题

### Q: 为什么我的电脑运行不了？
**A**: 95% 的情况是缺少 VC++ 运行时，安装上面的链接即可解决。

### Q: 会损坏我的电脑吗？
**A**: 完全不会。这只是纯截图工具，不包含任何恶意代码。

### Q: 为什么文件这么大（40MB）？
**A**: 因为打包了完整的图形界面库（PyQt5），这是正常大小。

### Q: 可以放 U 盘在其他电脑使用吗？
**A**: 可以！EXE 文件完全独立，复制到其他 Windows 电脑直接运行。

## 使用说明

1. **启动程序**：双击 `ScreenshotTool.exe`
2. **选择区域**：
   - 屏幕变暗，出现红色边框
   - 按住鼠标左键拖动选择区域
   - 松开鼠标完成截图
3. **预览窗口**：
   - 拖动窗口：按住标题栏拖动
   - 保存截图：点击"保存"按钮
   - 关闭窗口：点击"关闭"按钮

---

## 卸载

直接删除 `ScreenshotTool.exe` 即可，不会留下任何残留文件。
